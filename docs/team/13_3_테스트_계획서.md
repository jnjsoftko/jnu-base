## 1. 테스트 개요
### 1.1. 테스트 범위
```mermaid
graph TD
    A[테스트 영역] --> B[기능 테스트]
    A --> C[성능 테스트]
    A --> D[연계 테스트]
   
    B --> B1[DB CRUD]
    B --> B2[백업/복구]
   
    C --> C1[부하 테스트]
    C --> C2[스트레스 테스트]
   
    D --> D1[API 연동]
    D --> D2[배치 작업]
```
